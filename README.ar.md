# دورة بايثون

[English](README.md) · [Español](README.es.md) · [Català](README.ca.md) · [Svenska](README.sv.md) · العربية

## ما هو هذا الكتاب؟
دورة عملية لتعلّم بايثون خطوة بخطوة (مناسبة تقريبًا لعمر 14 سنة). كل فصل يحتوي على:
- هدف واضح
- أمثلة قصيرة يمكنك تشغيلها
- تحديات صغيرة مع TODOs وتلميحات
- أخطاء شائعة (حتى لا تعلق لفترة طويلة)

## كيف تستخدمه (بدون ضغط)
1. ابدأ بـ **الفصل 1** وجهّز جهازك.
2. اقرأ فصلًا واحدًا وشغّل الأمثلة.
3. نفّذ التمارين: الأخطاء جزء طبيعي من التعلّم.
4. إذا علِقت، ارجع إلى قسم **الأخطاء الشائعة** في ذلك الفصل.

## اختر مسارًا للتعلّم
يبقى الفهرس المرقّم هو ترتيب المرجع الثابت، وفيه روابط الفصول لكل مسار. يوضّح المسار ما هو مطلوب وأين يمكنك التوقف بأمان؛ المعاينات الاختيارية ومواد hero لا تمنع أبدًا نقطة التحقق الأساسية التالية. إذا كانت لديك خبرة، فافتح من الفهرس نقطة الإكمال المذكورة، ونفّذ التقييم الذاتي، وابدأ المسار التالي إن حققت معاييره.

- **الأساسيات الضرورية · 12–18 جلسة من 45–70 دقيقة.** المتطلبات: لا شيء. ابدأ من الفصل 1 وأكمل النقاط الأساسية حتى «نقطة التحقق وسلّم التقييم» في الفصل 12. النتيجة: بناء برنامج محلي صغير وشرحه باستخدام البيانات والقرارات والتكرار والدوال وصنف واحد. يمكنك التوقف بعد أي نقطة تحقق.
- **بايثون العملية · 6–10 جلسات من 50–80 دقيقة.** المتطلب: نقطة الفصل 12 الأساسية. تابع الفصول 13–18 وأنهِ «نقطة التحقق وسلّم التقييم» في الفصل 18. النتيجة: تنظيم مشروع محلي وحفظ بياناته واختباره وإعادة إنشائه. الفصل 18 نقطة توقف مكتملة.
- **بايثون للأنظمة · 5–9 جلسات إضافة إلى مختبر الشبكات.** المتطلب: نقطة الفصل 18. تابع الفصول 19–23 وأنهِ «سلّم التقييم» في الفصل 23. النتيجة: تشخيص برامج محلية محدودة للـHTTP والتسجيل والمهام غير المتزامنة والاستبطان والشبكات. يمكنك التوقف عند الفصل 22 قبل المختبر الطويل.
- **مسار hero اختياري Python + C++ · 12–18 جلسة.** المتطلبات: الفصول 15 و16 و18. اتبع الفصل 24 وأكمل مشروعه باستخدام سلسلة أدوات معلنة ومختبرة. النتيجة: بناء امتداد Python مضبوط الأنواع ومدعوم بـC++ والتحقق منه. Rust غير مطلوب.
- **مسار hero اختياري Python + Rust · 12–18 جلسة.** المتطلبات: الفصول 15 و16 و18. اتبع الفصل 25 وأكمل التحقق باستخدام سلسلة أدوات معلنة ومختبرة. النتيجة: بناء امتداد Python مضبوط الأنواع ومدعوم بـRust والتحقق منه. C++ غير مطلوب.
- **ملحقان مستقلان · 1–2 جلسة لكل منهما.** يبدأ ملحق CLI بعد الفصلين 18 و20. يبدأ ملحق الخوارزميات بعد الفصول 5 و7 و10 و18. لكل ملحق نقطة تحقق وتوقف خاصة به.
