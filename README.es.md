## Qué es este libro
Un curso práctico de Python pensado para aprender paso a paso (ideal para ~14 años). Cada capítulo tiene:
- Un objetivo claro
- Ejemplos cortos que puedes ejecutar
- Retos con TODOs y pistas
- Errores comunes (para no frustrarte)

## Cómo usarlo (sin estrés)
1. Empieza por el **Capítulo 1** y deja tu ordenador listo.
2. Lee un capítulo y prueba los ejemplos.
3. Haz los ejercicios: si algo falla, es parte del aprendizaje.
4. Si te atascas, vuelve a **Errores comunes** del capítulo.
