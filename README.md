## What this is
A practical, step-by-step Python course (great for ~14-year-olds). Every chapter includes:
- A clear goal
- Short runnable examples
- Small challenges with TODOs and hints
- Common mistakes (so you don’t get stuck for long)

## How to use it (no stress)
1. Start with **Chapter 1** and set up your computer.
2. Read one chapter and run the examples.
3. Do the exercises: errors are part of learning.
4. If you’re stuck, check **Common mistakes** in that chapter.
