## Què és aquest llibre
Un curs pràctic de Python per aprendre pas a pas (ideal per a ~14 anys). Cada capítol té:
- Un objectiu clar
- Exemples curts que pots executar
- Reptes amb TODOs i pistes
- Errors comuns (per no frustrar-te)

## Com fer-lo servir (sense estrès)
1. Comença pel **Capítol 1** i deixa l’ordinador a punt.
2. Llegeix un capítol i prova els exemples.
3. Fes els exercicis: si alguna cosa falla, forma part d’aprendre.
4. Si t’encalles, torna a **Errors comuns** del capítol.
